[PL]
Cały quiz zaprojektowany został w stylistyce systemów Windows 95/Windows 98.
Pulpit jest w pełni funkcjonalny i pozwala otwierać okna.
Pasek zadań zawiera funkcjonalne menu start, działający zegar, oraz pozwala na otwieranie zminimalizowanych okien.
Każdy element quizu mieści się w osobnym oknie które można dowolnie otwierać, zamykać, i minimalizować.

Autor: Antoni Przesławski
